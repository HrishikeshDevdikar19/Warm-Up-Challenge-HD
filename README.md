# Adaptive AI Tutor

## 🧠 Problem Statement
Many learners struggle to understand new concepts because explanations are not personalized to their level.

## 💡 Solution
We built an AI-powered tutor that:
- Explains concepts based on user input
- Asks follow-up questions
- Evaluates user responses
- Adapts explanations accordingly

## ⚙️ Tech Stack
- Frontend: React
- Backend: Node.js (Express)
- Future Scope:
  - Google Cloud Run (deployment)
  - Firebase (user progress storage)
  - Vertex AI (advanced AI responses)

## 🚀 Features
- Topic-based learning
- Interactive Q&A
- Adaptive feedback system
